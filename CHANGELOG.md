# Changelog

All notable changes to the personal portfolio site documented here.

## [0.1.0] — 2026-06-04

Initial public source release. Live at [miguelborges.dev](https://miguelborges.dev).

### Added

- Single-page static export (Next.js 14 with `output: 'export'`)
- Custom typography stack: Instrument Serif (display) · Geist (body) · JetBrains Mono (accents)
- Contact form via Formspree
- Responsive design, no JS animation libraries

### Stack

Next.js 14 · React 18 · TypeScript · Tailwind CSS 3 · static export

### Deployment

Built and deployed via Vercel from this repo's `main` branch.
